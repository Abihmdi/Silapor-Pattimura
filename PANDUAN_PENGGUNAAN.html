Bab 01
Apa itu KMP Pattimura?

KMP Pattimura adalah aplikasi web untuk pelaporan keuangan Koperasi Desa Merah Putih di seluruh wilayah Kepulauan Maluku. Aplikasi ini menggantikan sistem pelaporan via WhatsApp sebelumnya, sehingga proses pelaporan menjadi lebih terstruktur, terdokumentasi, dan mudah dipantau.

Bab 02
Cara Mengakses Aplikasi

Aplikasi KMP Pattimura bisa dibuka dari HP (smartphone) maupun laptop/PC — cukup buka browser dan ketik alamat website.

Buka browser apapun (Chrome, Safari, Firefox, Edge)
Ketik alamat: silapor-pattimura-xxx.vercel.app (tanyakan link terbaru ke admin)
Aplikasi langsung terbuka — tidak perlu install apapun
💡
Tips: Di Chrome HP, ketuk ⋮ → "Add to Home Screen" agar bisa buka seperti aplikasi biasa tanpa mengetik URL setiap kali.

Bab 04
Portal Koperasi — Mengirim Laporan

Setelah login sebagai koperasi, kamu akan melihat 2 tab: 📤 Kirim Laporan dan 📋 Riwayat.

4.1 Mengisi Form Laporan

Form laporan terdiri dari 5 bagian. Scroll ke bawah untuk mengisi semua bagian.

Bagian 1: 📌 Identitas Laporan

Periode
Bulan laporan yang dilaporkan. Contoh: Mei 2025
Jenis Laporan
Pilih: Bulanan, Triwulan, atau Tahunan
Jumlah Anggota Aktif
Total anggota koperasi saat ini. Contoh: 120

Bagian 2: ⚖️ Neraca (Posisi Keuangan)

Bagian ini melaporkan kondisi aset, utang, dan modal koperasi.

📋 Sub-bagian: Aset (Harta Koperasi)
Total Aset (Rp)
Jumlah seluruh harta koperasi. Masukkan angka tanpa titik, contoh: 450000000
Kas & Bank (Rp)
Uang tunai + saldo di rekening bank
Piutang Anggota (Rp)
Uang pinjaman yang belum dikembalikan anggota
Pinjaman Disalurkan (Rp)
Total pinjaman yang sudah diberikan ke anggota
📋 Sub-bagian: Kewajiban & Ekuitas
Total Kewajiban / Utang (Rp)
Utang koperasi ke pihak luar (bank, dll)
Dana Cadangan & Hibah (Rp)
Dana cadangan + hibah yang diterima
📋 Sub-bagian: Simpanan Anggota
Simpanan Pokok (Rp)
Dibayar sekali saat menjadi anggota — tidak bisa ditarik
Simpanan Wajib (Rp)
Dibayar rutin tiap bulan — tidak bisa ditarik
Simpanan Sukarela (Rp)
Tabungan anggota yang bisa ditarik kapan saja
⚠️
Cek Neraca Seimbang! Sistem otomatis menghitung:
• ✅ Hijau = Neraca seimbang (Aktiva = Pasiva) → Benar!
• ⚠️ Merah = Neraca belum seimbang → Cek ulang angka

Rumus: Total Aset = Kewajiban + Simpanan Pokok + Simpanan Wajib + Simpanan Sukarela + Dana Cadangan

Bagian 3: 📊 Perhitungan Hasil Usaha (PHU)

Bagian ini melaporkan pendapatan dan pengeluaran koperasi.

📋 Sub-bagian: Pendapatan (Pemasukan)
Jasa Pinjaman / Bunga (Rp)
Pendapatan bunga dari pinjaman anggota
Provisi & Administrasi (Rp)
Biaya admin yang dipungut dari anggota
Usaha Toko / Lainnya (Rp)
Pendapatan dari usaha toko atau unit usaha lain
Total Pemasukan (Rp) ⚡ Otomatis
Dihitung otomatis dari 3 field di atas — tidak perlu diisi manual
📋 Sub-bagian: Beban (Pengeluaran)
Beban Jasa Simpanan (Rp)
Bunga/jasa yang dibayar ke anggota atas simpanannya
Beban Operasional (Rp)
Gaji karyawan, listrik, sewa kantor, ATK, dll
Beban RAT & Koperasi (Rp)
Biaya rapat anggota, pelatihan, iuran koperasi
Total Pengeluaran (Rp) ⚡ Otomatis
Dihitung otomatis dari 3 field di atas — tidak perlu diisi manual
📋 Sub-bagian: Hasil Akhir
SHU Periode Ini (Rp) ⚡ Otomatis
= Total Pemasukan − Total Pengeluaran. Dihitung otomatis.
💡
Field bertanda ⚡ Otomatis (berwarna abu-abu) tidak perlu diisi manual. Sistem menghitung otomatis saat kamu mengisi field pendapatan dan beban di atasnya.

Bagian 4: 📝 Catatan & Rencana

Kendala yang Dihadapi
Masalah/hambatan operasional. Contoh: "Keterlambatan cicilan dari beberapa anggota"
Rencana Tindak Lanjut
Langkah yang akan diambil. Contoh: "Pendekatan personal ke anggota yang menunggak"
Catatan Tambahan
Info lain untuk pengawas. Contoh: "Simpanan meningkat 5% bulan ini"

Bagian 5: 📎 Lampiran File (Opsional)

Klik area "Ketuk untuk upload file"
Pilih file dari HP/komputer — format: PDF atau Excel (.xlsx/.xls)
File yang dipilih akan muncul di bawah area upload
Klik ✕ jika ingin menghapus file yang sudah dipilih
ℹ️
Upload file bersifat opsional. Laporan tetap bisa dikirim tanpa lampiran.

4.2 Mengirim Laporan

Bab 06
Fitur Tambahan

6.1 🖨️ Cetak Laporan ke PDF

Buka detail laporan (klik laporan manapun)
Klik tombol biru "🖨️ Cetak PDF" di bawah detail
Muncul dialog print dari browser
Pilih "Save as PDF" → klik Save

6.2 📥 Download Rekap ke Excel (CSV)

Masuk ke tab "📈 Rekap"
Atur filter periode dan kepulauan sesuai kebutuhan
Klik tombol hijau "📥 Unduh Rekap (Excel)"
File CSV terdownload otomatis — buka dengan Excel atau Google Sheets

File CSV berisi: Nama Koperasi, Pulau, Periode, seluruh data Neraca & PHU, Jumlah Anggota, Kendala, dan Rencana Tindak Lanjut.

6.3 ℹ️ Kamus Istilah Keuangan

Masuk ke tab "📈 Rekap"
Klik tombol biru "ℹ️ Bantuan Istilah"
Popup kamus istilah muncul dengan penjelasan: Total Aset, Kewajiban, Simpanan, SHU, Neraca

6.4 📄 Lihat Lampiran File

Buka detail laporan yang memiliki lampiran
Scroll ke bawah, cari bagian "Lampiran"
Klik link biru "📄 [Nama File] ↗" — file terbuka di tab baru

6.5 🔄 Mode Simulasi (Offline)

Jika koneksi internet terputus atau database sedang offline, aplikasi otomatis beralih ke Mode Simulasi dengan data contoh/demo.

🚨
Pada Mode Simulasi, data TIDAK tersimpan permanen. Laporan yang dikirim hanya tersimpan sementara dan hilang saat refresh. Pastikan kamu terhubung ke internet untuk mengirim laporan sungguhan.
Bab 07
FAQ & Troubleshooting
?
Password salah, tidak bisa login?
Koperasi: Pastikan kamu memilih koperasi yang benar dari dropdown, lalu masukkan password sesuai tabel di Bab 3.1.
Pengawas: Password pengawas adalah pengawas123 (huruf kecil semua, tanpa spasi).
?
Laporan tidak terkirim?
1. Pastikan minimal isi salah satu: Pemasukan, Jumlah Anggota, atau Catatan
2. Cek koneksi internet
3. Jika muncul "Supabase Offline", tunggu beberapa saat dan coba lagi
4. Jangan klik tombol kirim berkali-kali — tunggu proses selesai
?
Neraca menunjukkan "Belum Seimbang"?
Cek ulang: Total Aset harus sama dengan Kewajiban + Simpanan Pokok + Simpanan Wajib + Simpanan Sukarela + Dana Cadangan.

Neraca belum seimbang tidak menghalangi pengiriman laporan — ini hanya peringatan.
?
File lampiran tidak bisa dibuka oleh pengawas?
Ini masalah konfigurasi CORS pada storage. Sementara, koperasi bisa mengirim file lampiran via WhatsApp ke pengawas sebagai backup. Admin teknis perlu mengecek pengaturan di Supabase Storage.
?
Data tidak update / dashboard masih data lama?
Aplikasi tidak memiliki auto-refresh. Untuk melihat data terbaru:
• Klik ulang tab yang sedang aktif (contoh: klik "📊 Dashboard" lagi)
• Atau refresh halaman browser (tekan F5 atau tarik layar ke bawah di HP)
?
Apakah bisa digunakan di HP?
Ya! Aplikasi sudah mobile-friendly dan otomatis menyesuaikan ukuran layar HP. Tidak perlu setting apapun — cukup buka URL di browser HP.
?
Apakah data aman?
Data disimpan di Supabase (cloud database), bukan di HP/komputer kamu. Namun, aplikasi ini masih tahap prototipe — password disimpan plain text dan belum ada enkripsi. Jangan gunakan untuk data sangat sensitif sebelum ada peningkatan keamanan.

Butuh bantuan lebih lanjut?

Hubungi admin sistem atau pengawas pusat
